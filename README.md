# Scraper-Desktop-With-Python-Native-CLI

A simple desktop web scraper built with native Python CLI.

## Deskripsi

Proyek ini adalah aplikasi desktop sederhana untuk scraping data dari halaman web menggunakan Python CLI. Cocok untuk keperluan pembelajaran atau automasi pengambilan data ringan.

## Fitur

- Skrip utama: `scraper.py`.
- Export to : `.md,.docx,.txt`;
